=head1 NAME

size.pl - PyukiWiki / YukiWiki プラグイン

=head1 概要

 &size(ピクセル数値){文字列};
 &size(20){サイズ２０で表示};

=head1 説明

文字の大きさを指定する。

このプラグインはYukiWikiと互換性があります。

=head1 補足

旧バージョンの size.inc.pl は廃止しますので、最新の size.pl をダウンロードして、削除する様にして下さい。
